# GULP config

Small GULP config to make landing pages with SASS and ES6+ JS

____

## Packages
* gulp
* browser-sync
* del
* gulp-autoprefixer
* gulp-concat
* gulp-imagemin
* gulp-notify
* gulp-plumber
* gulp-sass
* sass
* webpack-stream

____

``` gulp ``` - to start project

``` gulp build ``` - to build project
